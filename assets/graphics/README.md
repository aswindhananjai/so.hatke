# Design Graphics

Place your design graphic images here as transparent PNGs.

## Current Graphics (from data/graphics.csv)
- `ace-of-spades.png` - The Ace of Spades
- `lightning-mcqueen.png` - Lightning McQueen
- `karan-aujla.png` - Karan Aujla Silhouette
- `sunflowers.png` - Sunflowers

## Image Specifications
- **Format**: PNG with transparency
- **Size**: 400x400px to 600x600px
- **Background**: Transparent (IMPORTANT!)
- **Style**: Bold, high contrast
- **Colors**: Vibrant, visible on dark backgrounds

## Tips
- Graphics will be overlaid on denim jackets
- Use bold outlines for visibility
- Test on both light and dark backgrounds
- Keep file size under 200KB for fast loading
